# Attendance App - Student Attendance Management System

## 📖 About The Project

Attendance App is an Android application developed using Java and Android Studio to simplify student attendance management for teachers. The application allows teachers to add students, manage class records, mark attendance, and track student attendance efficiently.

## 🚀 Features

* Add New Students
* View Student List
* Mark Daily Attendance
* Present/Absent Status Management
* Attendance Records Tracking
* Update Student Information
* Delete Student Records
* User-Friendly Dashboard
* Local Data Storage

## 🛠️ Built With

* Java
* Android Studio
* XML
* Database
* RecyclerView
* CardView
* Material Design Components

## 📱 Application Screens

* Splash Screen
* Dashboard Screen
* Add Student Screen
* Student List Screen
* Attendance Marking Screen
* Attendance Records Screen

## 📂 Project Modules

### Dashboard Module

Provides quick access to attendance-related features and student management.

### Student Management Module

Allows teachers to add, update, and delete student information.

### Attendance Module

Enables teachers to mark daily attendance and maintain attendance records.

### Database Module

Handles secure local storage of student and attendance data using Database.

## 🎯 Objectives

The primary objective of this project is to provide a simple and efficient solution for managing student attendance digitally, reducing manual paperwork and improving record accuracy.

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/Attendance-App.git
```

2. Open the project in Android Studio.

3. Sync Gradle dependencies.

4. Run the application on an Android Emulator or Physical Device.
